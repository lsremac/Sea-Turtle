How to Play

🎮 Controls

  • Arrow Keys / WASD: Move turtle
  • Space: Turbo boost (when available)
  • P: Pause game
  • M: Mute sound

🎯 Objectives

  • Collect trash to increase ocean health
  • Avoid hazards like fishing nets and sharks
  • Use power-ups to enhance abilities
  • Complete levels to unlock new areas

🌟 Power-ups

  • ⚡ Turbo: Increased speed
  • 🛡️ Shield: Temporary protection
  • 🔍 Magnet: Attract nearby trash

About TurtleQuest

TurtleQuest: Ocean Savior is an educational game that raises awareness about ocean pollution while providing an engaging gaming experience.

Features:

  • 🎮 Multiple levels with increasing difficulty
  • 🤖 AI-powered turtle companion with ocean facts
  • 🌍 Real-time global leaderboards
  • 📱 Progressive Web App (PWA) support
  • 🎵 Immersive sound design
  • 🎨 Beautiful particle effects and animations

Built with:

Phaser 3, Node.js, Socket.IO, PostgreSQL, OpenAI API
